# Amazon Clone (Frontend)

## 📌 Overview

This is a **frontend clone of Amazon.com** built using **HTML and CSS**. It replicates the basic layout of the Amazon homepage including the navbar, product sections, and footer.

## 🚀 Features

* Amazon-style navigation bar
* Search bar UI
* Product sections/cards
* Footer layout

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
